Installation

cd openrag-demo

pip install -r requirements.txt

python app.py

Configuration

Configure the application using environment variables: 


OPENAI_API_KEY=your_api_key
VECTOR_DB=qdrant
CHUNK_SIZE=1000
CHUNK_OVERLAP=200
EMBEDDING_MODEL=text-embedding-3-large

Document Processing Workflow

  1. Upload a document.
  2. Extract text.
  3. Split into chunks.
  4. Create embeddings.
  5. Store vectors.
  6. Perform semantic retrieval.
  7. Generate answers using the retrieved context.

Frequently Asked Questions

What file formats are supported?

PDF, HTML, DOCX, TXT, Markdown, CSV, and JSON.

Which embedding model is used?

The default embedding model is text-embedding-3-large.

Can I upload multiple documents?

Yes. Documents are indexed independently while sharing the same searchable vector database.

Does the system support citations?

Yes. Retrieved chunks can be returned with source references.
